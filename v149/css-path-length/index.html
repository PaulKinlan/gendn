CSS path-length

A CSS property that exposes the SVG pathLength presentation attribute to the CSS cascade. Set a normalised total length on any SVG geometry element so that stroke-dasharray offsets are percentages of your number, not the browser's computed length.

why it exists

SVG's pathLength attribute lets you rescale the coordinate system used by stroke-dasharray and stroke-dashoffset. If you set pathLength="100" on a circle and then write stroke-dasharray: 50 50, you get equal dashes regardless of the circle's actual circumference — no JavaScript needed to calculate 2πr. The classic trick is pathLength="1": all dash/offset values become fractions of 1, making "draw 70%" of a path as simple as stroke-dasharray: 0.7 0.3.

Before Chrome 149, pathLength could only be set as an SVG presentation attribute, bypassing the CSS cascade. You couldn't animate it with @keyframes, override it with a stylesheet, or drive it with a CSS custom property. The new path-length CSS property unlocks all of those scenarios.

Source: w3c/svgwg issue #773 and blink-dev Intent to Ship.

live example

This circle uses path-length: 1 in CSS, so stroke-dasharray: 0.7 0.3 means "70% filled, 30% gap" — regardless of the circle's radius.

.draw-circle {
  /* Normalise: all dash/offset values are fractions of 1 */
  path-length: 1;

  /* 70% dash, 30% gap — works for any radius */
  stroke-dasharray: 0.7 0.3;
  stroke-dashoffset: 0;

  animation: draw 2s linear infinite;
}

@keyframes draw {
  to { stroke-dashoffset: -1; }
}

recipes

Classic "draw on" animation — path-length: 1 trick

/* Attribute approach (always worked): */
/* <path d="..." pathLength="1" /> */

/* CSS approach (Chrome 149+): */
.draw-path {
  path-length: 1;
  stroke-dasharray: 1;
  stroke-dashoffset: 1;
  transition: stroke-dashoffset 0.6s ease;
}
.draw-path.visible {
  stroke-dashoffset: 0;
}

Animate path-length itself with a CSS custom property

@property --pl {
  syntax: "<number>";
  initial-value: 100;
  inherits: false;
}

.shape {
  path-length: var(--pl);
  stroke-dasharray: 50;
  transition: --pl 1s ease;
}
.shape:hover {
  --pl: 50; /* changes how long "50" dash units feel */
}

Override via stylesheet (was impossible with attribute only)

/* Base stylesheet */
.default-path { path-length: 100; }

/* Theme override — higher specificity wins */
.theme-compact .default-path { path-length: 1; }
