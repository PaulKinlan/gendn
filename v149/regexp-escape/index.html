RegExp.escape()

A new static method on RegExp that escapes all metacharacters in a string so it can be safely embedded in a regular expression pattern as a literal — ending the pattern of hand-rolled escaping utilities. Part of ECMAScript 2026 (TC39 Stage 4).

this API is documented on MDN

example

// Before: hand-rolled escape
function escapeRegex(s) {
  return s.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
}

// After: built-in
const query = "Hello. (World)";
const pattern = new RegExp(RegExp.escape(query), 'i');
pattern.test("hello. (world)"); // true
