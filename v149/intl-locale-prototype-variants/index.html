Intl.Locale.prototype.variants

A new accessor property on Intl.Locale instances that returns the BCP 47 variant subtags associated with the locale — for example the orthography year (1901, 1996) or historical period (emodeng) that the triple of language + region + script cannot uniquely identify.

this API is documented on MDN
