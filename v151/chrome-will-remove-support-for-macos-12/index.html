Chrome will remove support for macOS 12

Chrome 151 is the first release that requires macOS 13 Ventura or later. Chrome 150 is the last version to support macOS 12 Monterey. Users on macOS 12 will continue to run Chrome 150 but will stop receiving updates, including security fixes.

End of support Users running macOS 12 Monterey will be stranded on Chrome 150 when Chrome 151 ships. Their browser will continue to work but will no longer receive security, stability, or feature updates from Chrome 151 onwards.

what this means

Apple ended security support for macOS 12 Monterey in 2024. To align with Apple's support window, Chrome is removing support for macOS 12 in Chrome 151, scheduled for stable release on July 28, 2026.

Machines running macOS 12 that have Chrome installed will retain Chrome 150 and can continue using it, but Chrome will display a warning banner and will not receive further updates. Security vulnerabilities discovered after Chrome 150's release will not be patched on those machines.

Users who can update their Mac to macOS 13 Ventura or later should do so to continue receiving Chrome updates. Some older Mac hardware is not eligible for macOS 13 and will be permanently stuck on Chrome 150.
