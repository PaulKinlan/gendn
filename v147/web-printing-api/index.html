Web Printing API

A JavaScript API for querying local printers, submitting print jobs, and managing print options — available exclusively in Isolated Web Apps (IWAs). Targets remote/virtual desktop and kiosk scenarios where window.print() falls short.

Isolated Web Apps only The Web Printing API is not available on ordinary web pages. It is restricted to Isolated Web Apps, which require their own install and packaging process. IWAs are ChromeOS / enterprise focused. Do not use this API if you are building a general website or PWA.

why it exists

window.print() shows a browser-controlled print dialog and offers no control over printer selection, job settings, or status. In remote desktop and managed enterprise deployments, an IWA may need to silently print to specific local printers, stream print job status back to a server, or let users pick from available printers in an in-app UI. The Web Printing API exposes these capabilities through navigator.printing inside IWA contexts, following the Internet Printing Protocol (IPP) semantics for attribute names and semantics.

Source: WICG Web Printing explainer, blink-dev Intent to Ship, April 2026.

shape of the API

Entry point

The API is exposed via navigator.printing in IWA contexts:

Method / PropertyDescription
navigator.printing.getPrinters() Returns a Promise resolving to an array of WebPrinter objects representing locally available printers. Triggers a permission prompt on first call.

WebPrinter interface

MemberDescription
printerInfoObject with printer metadata: name, URI, description, state.
fetchAttributes()Returns the printer's IPP attribute set — supported media sizes, color modes, finishing options, etc.
printJob(document, jobAttributes)Submits a print job. document is a Blob (e.g. PDF); jobAttributes is an IPP-style attribute map.
Source: WICG Web Printing API spec and blink-dev thread, April 2026.

example

Listing available printers

// Only works inside an Isolated Web App
const printers = await navigator.printing.getPrinters();
for (const printer of printers) {
  console.log(printer.printerInfo.name, printer.printerInfo.state);
}

Submitting a silent print job

// Fetch a PDF and print it to the first available printer
const [printer] = await navigator.printing.getPrinters();
const attrs = await printer.fetchAttributes();

// e.g., print to A4 paper, color, 2 copies
const response = await fetch('/report.pdf');
const pdfBlob = await response.blob();

await printer.printJob(pdfBlob, {
  'media': 'iso_a4_210x297mm',
  'print-color-mode': 'color',
  'copies': 2,
});
console.log('Print job submitted');
